# Jogo de xadrez completo com as jogadas Rock, enPassant e Promoção implementadas!
## Aqui está um projeto de jogo de xadrez do curso de Java do Nélio Alves, referência como professor de Java e C#.

## Pude aprofundar mais um pouco sobre orientação a objetos, programação defensiva, uso do (if) para testar as jogadas permitidas.

### Certa vez vi no linkedin que um recrutador pediu para o candidato em questão implementar os movimentos da rainha, após fazer esse projeto consegui entender o por que desse teste, pois a Rainha pode mover para todos os lados do tabuleiro sem limite de casas, então nessa implementação mostra o domínio do penssamento lógico para testar todas as posições possíveis e também a mecânica do CHECK  e  CHECK MATE.
